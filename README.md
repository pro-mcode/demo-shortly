## Overview

Shortly is a link shortening web app that allows users to generate short links, manage them, and access additional features like pricing plans, blog content, and user authentication.

## Features

- Shorten links easily
- Manage and track shortened links
- Pricing plans for premium features
- Blog section for updates and tips
- User login and signup
- Fully responsive design
- Styled with TailwindCSS
- Built with modular React components

## Tech Stack

Frontend:

- JavaScript (ES6+)
- TailwindCSS
- HTML

Other Tools:

- Git
- Vite

## Installation

Clone the repository and install dependencies:

```bash
git clone https://github.com/pro-mcode/demo-shortly.git
cd weather-now
npm install
```

## Usage

```bash
npm run dev
npm run build
npm run preview
```
